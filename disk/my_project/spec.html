INTERFACE USAGE SPECIFICATION




(1) Syntax : help
Semantics : It displays the general syntax and function of all the commands.

(2) Syntax : fdisk
Semantics : It is used to create the disk (”disk.xfs”) or to format the disk if already created.

  • (3) Syntax : load <pathname>
    Semantics : Loads a relation from the host file system to the disk

  • (4) Syntax : export <xfs_filename> <pathname>
    Semantics: export files from the disk to the host filesystem.

  • (5) Syntax : rm <xfs_filename>
    Semantics : Remove the relation from the disk

  • (6) Syntax : ls
    Semantics : Displays the list of relations in the disk.

  • (7) Syntax : db
    Semantics : Display block allocation map

  • (8) Syntax : cat <xfs_filename> :
    Semantics : It displays the contents of a relation in the filesystem.

  • (9) Syntax : copy <start_block> <end_block> <unix_filename>
    • Semantics : Copies the contents of specified block(s) in the filesystem to the specified filepath

  • (10) Syntax : dump relcat
    Semantics : It copies the contents of relation catalog to an external file named relationcatalog.txt..

  • (11) Syntax : dump attrcat
    Semantics : It copies the contents of attribute catalog to an external file named attributecatalog.txt.

  • (12) Syntax : exit
    Semantics : Exit the interface
