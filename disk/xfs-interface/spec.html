INTERFACE USAGE SPECIFICATION




(1) Syntax: help
Semantics : It displays the general syntax and function of all the commands.

<--> (2) Syntax : fdisk
Semantics : It is used to create the xfs disk (”disk.txt”) or to format the xfs disk if already created.

  • (3) Syntax : import <filename>
    Semantics : Loads a relation from the host file system to the xfs disk

    • Click here for more details

  • (4) Syntax : export <relation name> <unix_filename>
    Semantics: Export relation from the disk to the host filesystem.

  • (5) Syntax : rm <relation name>
    Semantics : Remove the relation from the xfs disk

  • (6) Syntax : ls
    Semantics : Displays the list of relations in the xfs disk.

  • (7) Syntax : db
    Semantics : Dump block allocation map

  • (8) Syntax : dump <block no> <unix_filename > :
    Semantics : Dump the contents of given block number in the specified file.

  • (10) Syntax : dump relcat
    Semantics : It copies the contents of relation catalog to an external file named relationcatalog.txt..

  • (11) Syntax : dump attrcat
    Semantics : It copies the contents of attribute catalog to an external file named attributecatalog.txt.

  • (12) Syntax : exit
    Semantics : Exit the interface
